# [Course Code] - Data Structures (C Language)

This repository contains algorithms and laboratory works implemented in C for the **COM2067(Data Structures)** course.

## 📚 Resources
The main textbook used is **"Data Structures Using C" by Reema Thareja**. Many examples in this repository are solutions, algorithms, or adaptations of the exercises found in this book.

## 📂 Structure

* **`exercises/`:** Solutions to end-of-chapter problems and algorithm implementations from the Reema Thareja textbook.
* **`labs/`:** Practical assignments and time-constrained problems solved during laboratory sessions.

## 🧠 Key Concepts
This repository includes practices on:
* **Linear Data Structures:** Linked Lists (Singly, Doubly, Circular), Stacks, and Queues.
* **Non-Linear Data Structures:** Trees (Binary Search Trees, Heaps) and Graphs.
* **Algorithms:** Various Searching and Sorting algorithms.
* **Memory Management:** Advanced pointer manipulation and dynamic memory allocation required for complex data structures.

## ⚙️ Compilation
The codes are compatible with the standard GCC compiler.
